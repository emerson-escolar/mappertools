# mappertools


## Dependencies
mappertools requires:

  - python
  - numpy
  - scipy
  - scikit-learn
  - pandas
  - networkx
  - pyclustering
  - matplotlib

## Install
Install from source:

```
git clone https://github.com/emerson-escolar/mappertools
cd mappertools
pip install -e .
```
